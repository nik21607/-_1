Bootstrap (также известен как Twitter Bootstrap) – свободный набор инструментов для создания сайтов и веб-приложений.Включает в себя HTML и CSS шаблоны оформления для типографики, Web-форм, кнопок, меток, блоков навигации и прочих компонентов Web-интерфейса, включая JavaScript-расширения.

Bootstrap использует самые современные наработки в области CSS и HTML, поэтому необходимо быть внимательным при поддержке старых браузеров.
